Reflection

Throughout the CSIS1430 Course, I began to strengthen soft skills such as critical thinking, resilience, creative troubleshooting, and active learning. All are skills that are key to becoming a fantastic developer because of the amount of stress that follows while engineering solutions to an impacting problem. The reintroduction of HTML and CSS kickstarted a faint but not entirely lost passion for creating digital media. While working through the JavaScript projects, since I began to notice a growing fluency that was not present in the past. This newfound discovery opened myself to the desire to learn as much as possible within the realms of development.

HTML

One of the most valuable lessons that I learned from this class was HTML is considered to be a descriptive language. Until that moment, my previous work clobbered between HTML and CSS in a butchered format. After a clear explanation of how to properly apply HTML descriptively, my mindset stood corrected forever. In past experiences, my approach directly referenced HTML tags by using ID or classes. Quite often, I found myself using tags in a non-standard format. This approach made factoring difficult for myself and others in the future.

CSS

The most valuable lessons involving cascading stylesheets are the implementation of flexbox and specificities. Years before this course, my understanding of specificity was poor. Due to the poor understanding of specificity, it was a continuous battle when drafting designs. Most of the prior patches were driven by forum snippets that never clearly explained what was happening and how to fix it properly. The instruction and guidance provided in this course have rectified past puzzling thoughts and future mental agony.

JavaScript

Before this course, I had no understanding of how to program JavaScript properly. The opinions fed by influences in the past stated JavaScript was terrible, and nobody uses it. This course completely changed my outlook on JavaScript's capabilities. It was a handy tool that I avoided for years. One of the most valuable topics touched in this course was JavaScript's DOM. The ability to hook an HTML Document to JavaScript and modify document elements in real-time is a compelling feature.

Final Thoughts

This course was a fantastic introduction back into a world I previously left in the mid-2000s. I struggled with the lack of constructive criticism that would help reduce potential bad habits. Beyond lacking feedback, the resources provided helped expand my knowledge. The direction provided a conduit to continue learning after this course ends.
